"Трекер прочитанных книг" - консольное приложение, разработанное для читателей, которые хотят хранить информацию о прочитанных ими книгах.

Для запуска, введите в консоль 'python main.py'

Работа приложения:
  При запуске вас встречает текстовое меню с подсказками.
  При вводе цифры 1 необходимо указать всю информацию о прочитанной книге, включая автора, название, вашу оценку и дату.
  При вводе цифр 2,3 и 4 выводится список всех записанных книг, их средняя оценка и статистика по авторам соответственно.
  При вводе цифры 5 выводится список книг, необходимо указать индекс книги для удаления, после чего она будет убрана из списка.
  Наконец, при вводе цифры 6 работа консольного приложения останавливается.

  Список автоматически сохраняется при изменении.
